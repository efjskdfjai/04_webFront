목록 관련 태그

ul 태그 (Unordered List) 태그 : 순서가 없는 목록
ol 태그 (Ordered List) : 순서가 있는 목록
li 태그 (List item) : ul or ol 태그 내부에 들어가 목록에 요소를 나타내는 태그

    type="a" -> 소문자 알파벳

    type="i" -> 소문자 로마숫자
    type="I" -> 대문자 로마숫자
    type="1" -> 숫자(기본값)
    start 속성 : 시작 번호 지정
